# ビジネス分析手法

## 1. PEST分析

PEST分析は、外部環境を4つの要因から分析する手法。自社・業界を取り巻く外部環境が、現在もしくは将来的にどのような影響を与えるかを把握・予測するための戦略的分析・市場調査フレームワーク。
モデルの要素は、その業界とそれが生産する商品に基づいて、特定の企業にとって重要性が異なる。たとえば、B2C企業やB2B企業は社会的要因の影響をより受けやすく、防衛請負業者は政治的要因の影響をより受けやすい。さらに、企業にとって将来変更される可能性が高い、より関連性のある要因は、より重要となる。例えば多額の借金をしている企業は、経済的要因（特に金利）により大きな注意を払う必要がある。

1. Political（政治的要因）：政府が経済にどのように介入するか。さらに政府は国の健康、教育、インフラに大きな影響を与える。
2. Economic（経済的要因）：経済成長、為替レート、インフレ率、および金利がどのように影響するか。
3. Social（社会的要因）：文化的側面と健康意識、人口増加率、年齢分布、キャリア態度、安全性などがどのように影響するか。
4. Technological（技術的要因）： 研究開発活動、自動化、技術的インセンティブ、技術的変化の速度などの技術的側面の影響について。

分析をPESTLE分析またはPESTEL分析に拡張すると、次のものが追加される。

5. Legal（法的要因）：、差別法、消費者法、独占禁止法、雇用法、安全衛生法など。
6. Environment（環境要因）：天候、気候、気候変動などの生態学的および環境的側面。

手順：
1. 各要素について、関連する要因をリストアップする
2. 各要因が事業にどのような影響を与えるか分析する
3. 重要度や影響度を評価し、優先順位をつける
4. 分析結果に基づいて、戦略や対策を検討する

## 2. 5Forces分析

業界の競争状況を5つの要因から分析する手法。自社の強みや課題を明確化して、脅威に対して有効な対策とポジショニングを考えるためのフレームワーク。

1. 既存競合との競争：自社を含む競合他社の知名度やブランド力、資金力、業界の規模や成長率などの要因について分析を行う。
2. 新規参入者の脅威：参入ハードル、参入者の技術レベル、ブランド力、可能性などについて分析する。
3. 代替品の脅威：業界の製品を代替するような可能性を持つ製品との競争関係、ニーズについて分析を行う。
4. 買い手（顧客）の交渉力：価格競争、競合他社の状況、値下げ幅を含めた自社の価格設定（無理な値下げ競争をしていないか）などの分析。
5. 売り手（サプライヤー）の交渉力：売り手の数、力関係、供給元を乗り換える際の手間とコストなどについての分析。

手順：
1. 分析対象の業界を明確にする
2. 各要因について、現状と将来の見通しを評価する
3. 各要因の強さを定性的または定量的に評価する
4. 5つの要因の相互作用を考慮し、全体的な競争環境を分析する
5. 分析結果に基づいて、戦略的な対応を検討する

## 3. SWOT分析

組織の内部環境と外部環境を4つの要素から分析する手法。意思決定プロセスの予備段階で多くの種類の組織の戦略的地位を評価するためのフレームワーク。

1. Strengths（強み）：目標達成に貢献する組織（個人）の特質。どのように強みを活かすか？
2. Weaknesses（弱み）：目標達成の障害となる組織（個人）の特質。どのように弱みを克服するか？
3. Opportunities（機会）：目標達成に貢献する外部の特質。どのように機会を利用するか？
4. Threats（脅威）：目標達成の障害となる外部の特質。どのように脅威を取り除く、または脅威から身を守るか？

手順：
1. 各要素について、関連する項目をリストアップする
2. リストアップした項目を評価し、重要度の高いものを選別する
3. 選別した項目を2x2のマトリックスに配置する
4. 各象限の関係性を分析し、戦略的な示唆を導き出す

回避すべき誤り：
1. 目標が定まっていない分析結果には意味がない。
2. SWOT項目は戦略を定めるものではなく、価値ある戦略を生み出すために状況を説明するものである。

## 4. TOWS分析

SWOT分析の結果を活用し、より具体的な戦略を立案する手法。

手順：
1. SWOT分析を実施し、結果を整理する
2. 以下の4つの戦略を検討する：
   - SO戦略（強みを活かして機会を活用）
   - ST戦略（強みを活かして脅威に対抗）
   - WO戦略（弱みを克服して機会を活用）
   - WT戦略（弱みを最小限に抑え、脅威を回避）
3. 各戦略について具体的なアクションプランを立案する
4. 優先順位を付けて実行計画を策定する

## 5. 3C分析

自社の市場における立ち位置（ポジション）や将来のリスクを把握し、最適なマーケティング戦略を立案することを目的とした分析手法。

1. Customer（顧客）
2. Competitor（競合）
3. Company（自社）

手順：
1. 各要素について以下の項目を分析する：
   - Customer：ニーズ、購買行動、セグメント等
   - Competitor：強み、弱み、戦略、市場シェア等
   - Company：強み、弱み、独自の価値提案等
2. 各要素の相互関係を分析する
3. 分析結果に基づいて、自社の強みを活かし、顧客ニーズに応える戦略を立案する

## 6. STP分析

市場を細分化し、狙うべき市場を定め、競合との差別化を図ることで効果的なマーケティング戦略を立案するためのフレームワーク。

1. Segmentation（セグメンテーション）
2. Targeting（ターゲティング）
3. Positioning（ポジショニング）

手順：
1. Segmentation：市場を意味のある顧客グループに分類する
2. Targeting：最も魅力的で収益性の高いセグメントを選択する
3. Positioning：選択したターゲット市場で、競合と差別化された独自のポジションを確立する

## 7. 4P→4C＋4E分析

マーケティングミックスの考え方を、顧客視点と体験視点に拡張した分析手法。

4P：
1. Product（製品）
2. Price（価格）
3. Place（流通）
4. Promotion（プロモーション）

4C：
1. Customer Value（顧客にとっての価値）
2. Cost（顧客のコスト）
3. Convenience（利便性）
4. Communication（コミュニケーション）

4E：
1. Experience（体験）
2. Exchange（交換）
3. Everyplace（どこでも）
4. Evangelism（布教）

手順：
1. 4Pの各要素について現状を分析する
2. 4Pを4Cの視点で再解釈し、顧客視点での課題や機会を特定する
3. さらに4Eの視点を加え、顧客体験全体を考慮した分析を行う
4. 分析結果に基づいて、顧客中心のマーケティング戦略を立案する

## 8. 7P分析

サービス業向けに拡張されたサービスマーケティングミックスの分析手法。

無形性/非有形性：サービスは形がなく無形財である。
同時性/不可分性：サービスは生産と消費が同時に発生する。
非均一性/変動性：サービスは誰がいつ提供するかで品質を標準化することが困難である。
消滅性/非貯蔵性：サービスは蓄えることができない。

1. Product（商品・サービス）
2. Price（価格・プライシング）
3. Place（流通・チャネル）
4. Promotion（販促・プロモーション）
5. People（人材・要員）
6. Process（業務プロセス・販売プロセス）
7. Physical Evidence（物的証拠）

手順：
1. 各要素について現状を詳細に分析する
2. 各要素の強みと弱みを特定する
3. 競合他社との比較分析を行う
4. 顧客ニーズとの適合性を評価する
5. 改善が必要な領域を特定し、具体的な施策を立案する

## 9. VRIO分析

企業の内部資源を分析し、持続的競争優位性を評価する手法。VRIO分析を正しく活用できると、自社の強み・弱みを把握した上で効果的な経営戦略が策定できる。

1. Value（価値）
2. Rarity（希少性）
3. Imitability（模倣困難性）
4. Organization（組織）

手順：
1. 分析対象となる経営資源をリストアップする
2. 各資源について、以下の質問に答える：
   - Value：顧客に価値を提供し、脅威を中和できるか？
   - Rarity：競合他社は同様の資源を保有していないか？
   - Imitability：競合他社が模倣するのは困難か？
   - Organization：その資源を活用する組織能力があるか？
3. 各質問への回答に基づいて、資源の競争優位性を評価する
4. 分析結果に基づいて、持続的競争優位を構築するための戦略を立案する

## 10. バリューチェーン分析

商品を作るための原材料の調達から市場での流通・販売までの流れを企業の提供する「価値の連鎖」として考え、事業の工程ごとに分析する手法。企業の活動を価値創造の観点から分析する。

主活動：
1. 購買物流
2. 製造
3. 出荷物流
4. マーケティング・販売
5. サービス

支援活動：
1. 全般管理
2. 人事・労務管理
3. 技術開発
4. 調達活動

手順：
1. 自社のバリューチェーンを図示する
2. 各活動の詳細を記述し、コストと価値創造への貢献を分析する
3. 業界のバリューチェーンを分析し、自社の位置づけを確認する
4. 競合他社とのバリューチェーンを比較分析する
5. 改善可能な領域を特定し、競争優位を構築するための戦略を立案する

## 11. なぜなぜ分析

複数回の「なぜ？」を問いアイデアを掘り下げ、問題の根本原因を特定するための分析手法。事象が起きた原因だけでなく、さらにその原因に対しても「なぜ？」を繰り返し問い続ける。

手順：
1. 問題を明確に定義する
2. 「なぜ」という質問を5回程度繰り返す
3. 各「なぜ」に対して、できるだけ具体的な回答を用意する
4. 回答の連鎖から、根本原因を特定する
5. 特定した根本原因に対する対策を立案する

## 12. シックスハット法

6つの思考モードを使い分けて、多角的な視点で問題解決や意思決定を行う手法。

1. 白色帽子：客観的・中立的な視点。客観的な事実と数値、情報を用いた意見。
2. 赤色帽子：主観的・直感的な視点。直感的にどのように思ったかを感情や主観で表現する。
3. 黒色帽子：客観的・否定的な視点。論理的に否定的な意見で潜在的な問題やリスクを探る。
4. 黄色帽子：客観的・肯定的な視点。論理的に肯定的な意見でメリットや可能性を考える。
5. 緑色帽子：創造的・革新的な視点。事実やデータにとらわれない新しい選択肢や新しいアイデアを模索する。
6. 青色帽子：統括的・俯瞰的な視点。最後にアイデアを統括してまとめる。

手順：
1. 分析対象となる問題や課題を明確にする
2. 各色の帽子を順番に被り、その視点からの意見や考えを出し合う
3. 全ての帽子を一巡させた後、出された意見を整理する
4. 必要に応じて特定の帽子に焦点を当てて深堀りする
5. 多角的な視点からの分析結果に基づいて、最適な解決策や意思決定を行う

## 13. オズボーンのチェックリスト

創造的思考を促進するための9つの質問からなるチェックリスト。

1. 転用（例：他に使い道はないか？他に活躍できる分野・領域はないか？）
2. 応用（例：他の業種・分野からアイデアを借りられないか？）
3. 変更（例：形・機能を変えてみたらどうか？）
4. 拡大（例：時間・頻度・高さ・長さ・強さなどを拡大してみたらどうか？）
5. 縮小（例：携帯化できるか？短くできるか？軽くできるか？省略できるか？）
6. 代用（例：他のもので代用できないか？）
7. 置換（例：要素・パターン・配列・ペース・スケジュールなどを変えられないか？）
8. 逆転（例：マイナスをプラスにできないか？役割を逆にできないか？）
9. 結合（例：組み合わせてみたらどうか？ 目的や考えを結合できないか？ ）

手順：
1. 改善や革新が必要な対象（製品、サービス、プロセスなど）を選定する
2. 各質問に対して、できるだけ多くのアイデアを出す
3. 出されたアイデアを評価し、実現可能性や効果が高いものを選別する
4. 選別されたアイデアをさらに具体化し、実行計画を立てる

## 14. マンダラート

アイデア発想法の一つで、中心テーマを周囲に展開していく手法。連想からアイデアを引き出していくことが重視されており、「引き出された言葉に対して、新たなマンダラートを作成し、徐々にアイデアの裾野が広がっていく」という点が特徴。

手順：
1. 3x3のマス目を用意し、中央に主要テーマを書く
2. 周囲の8マスに、主要テーマに関連するサブテーマを書く
3. 各サブテーマを中心とした新たな3x3のマス目を作成する
4. 新しいマス目の周囲8マスに、さらに関連するアイデアを書く
5. この過程を繰り返し、アイデアを拡張していく
6. 完成したマンダラートを俯瞰し、新しい関連性や可能性を探る

## 15. 希望点列挙法、欠点列挙法

「こうすればより良くなる」「この点に不満を持っている」というアイデアをなるべく多く書き出し、対象の長所や短所を徹底的に洗い出す発想法。

手順（希望点列挙法）：
1. 分析対象を明確にする
2. 対象の理想的な状態や望ましい特徴を自由に列挙する
3. 列挙された希望点を分類・整理する
4. 各希望点の実現可能性や重要度を評価する
5. 優先順位の高い希望点から、実現のための方策を検討する

手順（欠点列挙法）：
1. 分析対象を明確にする
2. 対象の問題点や改善が必要な点を自由に列挙する
3. 列挙された欠点を分類・整理する
4. 各欠点の深刻度や改善の緊急性を評価する
5. 優先順位の高い欠点から、改善策を検討する

## 16. エクスカーション法

問題解決や創造的思考を促進するために、現実の制約から一時的に離れて自由な発想を行う。「あるモノの特徴」と「アイデア出しを行いたいテーマ」を掛け合わせることで、アイデアを連想していく手法。

手順：
1. 現実の問題や課題を明確にする
2. 「もし〜だったら」という仮想的な状況を設定する（例：「もし無限の予算があったら」「もし魔法が使えたら」）
3. 設定した仮想状況下で、自由にアイデアを出し合う
4. 出されたアイデアを整理し、現実世界での実現可能性を検討する
5. 実現可能なアイデアを選別し、具体的な行動計画に落とし込む

## 17. NM法

アイデア出しを行いたいテーマに関して、すでに世の中に存在する類比（異なる2つ以上の事象に間に存在する、何らかの同一性のこと）の例を探し出し、その特徴から発想していく手法。

NM法には、以下の5種類が存在します。
1. NM-H型：画期的な発明をするために着想と発想を行う方法
2. NM-T型：雑多な情報からテーマに関するヒントを得る方法
3. NM-S型：過去・現在・未来という時系列の中でアイデアを実用化する方法
4. NM-A型：書き出したアイデアをグルーピングし、新たなアイデアを得る方法
5. NM-D型：仮説を立て、データで証明するようにアイデア出しを行う方法

最も一般的に使用されているNM-H型の手順：
1. テーマのキーワードを考える：例えば、新しいスマートフォンケースのアイデア出しを行う場合、キーワードの主な候補は「守る」「持ち運ぶ」「取り付ける」「取り外す」など。
2. キーワードと似ているモノを見つける：すでに世の中に存在するキーワードに似ているモノ（類比）を見つける。連想ゲームのように「◯◯といえば何？」と自身に問いかける。
3. 似ているモノの特徴・背景を洗い出す：見つけた類比の特徴や背景を洗い出す。
4. テーマと似ているモノをつなげる：類比するモノの仕組みがアイデア出しを行いたいテーマに対してどのように役立つかを考える。
5. アイデアのクオリティを高める：アイデアを具体的・現実的にするために、煮詰めていく。また、アイデア同士を組み合わせ予想外のアイデアを生み出す。

## 18. 逆張り思考法

常識となっている物事について、あえて逆の思考・行動を選択した場合にどうなるかをイメージする手法。

手順：
1. 固定観念や先入観を書き出す。
2. 次にその固定観念や先入観に対して逆張り（逆の思考・行動）を検討する。常識を取り払い、出来るだけ自由で物事の根源に迫りながら逆張りをする。
3. 逆張りを具体化する。ここでは、多様な発想・珍しさ・新しさを重視する。
